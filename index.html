Privacy & Dati
💾 Dati sul dispositivo
Tutti i tuoi dati (cani, pedigree, cucciolate, profilo) vengono salvati esclusivamente sul tuo dispositivo tramite localStorage. Nessun dato viene trasmesso a server esterni.
🤖 Gemini AI (opzionale)
La lettura OCR del pedigree utilizza le API di Google Gemini. Le immagini vengono inviate ai server Google solo quando richiedi esplicitamente la lettura AI.
📱 Nessun account richiesto
L'app funziona completamente offline, senza registrazione.
